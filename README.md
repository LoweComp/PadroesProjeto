# 🎭 PadroesProjeto - DCC078 (João Victor Dias 25.3)

Repositório dedicado ao estudo e implementação prática de **20 Padrões de Projeto (Design Patterns)**, desenvolvidos durante o semestre **25.3** para a disciplina **DCC078 - Aspectos Avançados em Engenharia de Software**.

## 📌 Sobre o Projeto
O objetivo deste projeto é aplicar os conceitos de arquitetura de software em cenários reais do mercado de **produção de eventos** e da **classe artística**, com foco especial em:
* 🎸 **Música** (Shows, festivais e produção fonográfica).
* 🎬 **Cinema** (Produções audiovisuais e gestão de clássicos).
* 🎫 **Gestão de Eventos** (Promoters, casas de show e bilheteria).
